Наш алгоритм!

  • Слово может иметь 4 статуса, которые проходит за другом: не выучено → изучается → почти выучено → выучено. Каждый раз, когда пользователь правильно отгадывает слово на странице изучения слов или в любой из игр, слову присваивается статус выше предыдущего на один пункт. Если же он ошибается - слову присваивается статус "не выучено".
  • Список слов для изучения формируется из случайных слов, в которые добавляются слова из словаря. Слово добавляется к изучению, если с последнего изменения его сложности прошло определенное время. Для статуса "почти выучено" - три дня, для статуса "изучается" - один день, а слова со статусом "не выучено" добавляются к изучению сразу.
  • Если во время изучения пользователь ошибся в слове, слову присваивается статус "не выучено" и оно еще раз встретится в этой же игре.
